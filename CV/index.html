GitHub: όχι μόνο για κομπιουτεράδες

Το GitHub είναι κοινωνικό δίκτυο αλλά δεν έχει καμία σχέση με αυτό που φαντάζονται οι περισσότεροι όταν σκέφτονται το Facebook: δεν υπάρχουν φωτογραφίες και βίντεο, τα μέλη του δεν εκθέτουν τις προσωπικές και κοινωνικές τους στιγμές ή τις μεγάλες αλλαγές της ζωής τους και οι εμπορικές εταιρείες δε διαγκωνίζονται για να αναρτήσουν εκεί τις διαφημίσεις τους. Όμως το GitHub είναι από μια άποψη πολύ πιο ουσιαστικά «κοινωνικό» καθώς τα 3 εκατομμύρια μέλη του ανακοινώνουν projects ζητώντας τη βοήθεια της κοινότητας για να τα ολοκληρώσουν. Και αν κρίνουμε από τη δημοτικότητα του μοντέλου (το χρησιμοποιούν σαν πλατφόρμα επικοινωνίας για τα στελέχη τους εταιρείες όπως η Microsoft και η Google), μάλλον λειτουργεί πολύ καλά

Αυτό που κάνει το GitHub πετυχημένο ήταν ότι στήθηκε έτσι που να ευνοεί τη συνεργασία μεταξύ προγραμματιστών –ακόμα και σήμερα, η πλειονότητα των χρηστών του προέρχονται από αυτόν το χώρο. Όμως, καθώς το μοντέλο βάσει του οποίου εργάζονται (και συνεργάζονται) οι προγραμματιστές και ειδικά αυτοί του λεγόμενου «ανοιχτού κώδικα», δηλαδή αυτοί που δεν περιορίζουν τη δουλειά τους για λόγους πνευματικών δικαιωμάτων αλλά την αφήνουν ελεύθερη στο Δίκτυο, μπορεί να λειτουργήσει και για μια σειρά από άλλους χώρους. Και ήδη βρίσκει κανείς στο σάιτ, πρότζεκτ σχετικά με (σχεδόν) ότι μπορεί να φανταστεί: από νομικά έγγραφα μέχρι σχέδια βελτίωσης σπιτιού και μέχρι μουσικά έργα.

Βλέποντας αυτό το εύρος εφαρμογών της βασικής ιδέας του GitHub δεν είναι δύσκολο να σκεφτεί κανείς πόσο μπορεί να βοηθήσει στην εξέλιξη πρότζεκτ που σχετίζονται με τον πολιτισμό: αν και η δημιουργία δεν παύει να είναι, κατά βάση, ατομική δραστηριότητα, η συνδυασμένη δημιουργικότητα περισσότερων του ενός ανθρώπων είτε σε ένα κοινό έργο, είτε σε μια δράση είναι κάτι που έχει δοκιμαστεί και αποδειχθεί αποτελεσματικό επί αιώνες. Αν ο κόσμος του πολιτισμού υιοθετήσει το GitHub και τους τρόπους που διευκολύνει τη συνεργασία και τους εφαρμόσει σε αντίστοιχα έργα, είναι βέβαιο ότι τα αποτελέσματα που θα δούμε θα είναι εξαιρετικά ενδιαφέροντα.

Τι είναι το jekyll

ο jekyll είναι ένας τρόπος να αναπαράγουμε εύκολα και γρήγορα στατικές σελίδες ενός site χωρίς να χρειάζεται να επαναλαμβάνουμε τον ίδιο κώδικα σε κάθε μια απ' αυτές. Αυτό επιτυγχάνεται δημιουργώντας ένα συγκεκριμένο layout σε κάποιο φάκελο και στη συνέχεια το "καλούμε" σε όποια σελίδα θέλουμε να το χρησιμοποιήσουμε.

Σ' ένα συγκεκριμένο φάκελο δημιουργούμε το αρχείο το οποίο περιέχει το βασικό κώδικα της σελίδας μας χωρίς φυσικά το περιεχόμενο π.χ. Default_Layout.html. Στα σημεία που θα θέλουμε να τοποθετήσουμε το δικό μας περιεχόμενο χρησιμοποιούμε τα λεγόμενα liquid taqs στα κατάλληλα σημεία π.χ. {{ content }} και {{ page.title }}

Στην συνέχεια στο αρχείο index.html της κάθε σελίδας μας αντί να αναπαράγουμε το πλήρη html κώδικα καλούμε το jekkyll να επεξεργαστεί το αρχείο μας με τη χρήση κάποιων γραμμών απλού κειμένου στην αρχή του αρχείου ( front-matter). π.χ.:
